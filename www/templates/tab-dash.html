This starter --- extends the Tab based starter with a Map Tab, and controller

Modifications made to original Tab Starter

  • Uses gulp-inject to manage includes with gulp-inject as described at http://digitaldrummerj.me/gulp-inject/
  • Split controllers into unique files

This was based on the original Ionic starter for tabs-based apps. For other starters and ready-made templates, check out the Ionic Market or the GitHub Repo at https://github.com/driftyco/ionic-starter-tabs

To edit the content of each tab, edit the corresponding template file in www/templates/. This template is www/templates/tab-dash.html
